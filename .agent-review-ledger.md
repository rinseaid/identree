## Deferred Items (Require User Decision)

### Architecture

1. **O2/P7: Store context propagation** — All SQLStore methods use `context.Background()` instead of request context. Refactoring to accept `context.Context` prevents DB connection exhaustion under cancellation storms. Medium-effort change touching all handlers and the Store interface.

2. **O3: Deploy goroutine shutdown** — `runDeployJob` doesn't select on `s.stopCh`, so SSH deploy sessions continue running up to 3 minutes after SIGTERM. Requires context cancellation + WaitGroup drain in `Stop()`.

3. **O4: Audit sink retry** — Splunk HEC and Loki sinks discard batches on HTTP failure with no retry. For a hash-chained audit trail, this creates integrity gaps. Needs retry loop with backoff in `flush()`.

4. **O5: Request correlation IDs** — No request ID links log entries across a single HTTP request lifecycle. Needs middleware to generate/propagate IDs via context.

5. **O8: Health probe parallelization** — `/healthz` probes run sequentially (~6s when all time out). K8s liveness probes with <6s timeout will false-positive restart pods. Should run concurrently via errgroup.

6. **P4/P5: AllActionHistoryWithUsers pagination** — Admin history endpoint returns entire action_log table with no pagination. Needs Store interface change to add limit/offset.

### UI/UX

7. **A1: Grid column filter aria-labels** — ~20 filter inputs across grid tables use only `placeholder` for labeling. Each needs a unique `aria-label` derived from its column header. Could be done via JS reading the column header text.

8. **A2+A4: Keyboard support + focus trapping** — Custom `role="button"` elements and modal dialogs lack proper keyboard interaction (focus trap, Escape to close, Tab cycling). Significant JS work.

9. **A3: Replace confirm() dialogs** — Browser `confirm()` calls are inaccessible to some assistive technologies. Replacing with custom modals requires ARIA alertdialog implementation.

10. **A7: Color contrast audit** — Some semantic color variables (especially `--text-2` on default backgrounds) may not meet WCAG 2.1 AA 4.5:1 contrast ratio. Needs automated contrast testing tool.
